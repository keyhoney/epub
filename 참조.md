
## 1. 기본 텍스트 편집 및 스타일 (Essential)
가장 자주 쓰이는 텍스트 서식 도구입니다. 리플로우(Reflowable) 환경을 깨뜨리지 않도록 인라인 스타일 대신 시맨틱 태그와 CSS 클래스로 적용되게 구현해야 합니다.

* 글꼴 스타일: 굵게(<b>/<strong>), 기울임(<i>/<em>), 밑줄(<u>), 취소선(<s>)
* 텍스트 정렬: 왼쪽, 가운데, 오른쪽, 양쪽 정렬 (전자책 가독성을 위해 양쪽 정렬 필수)
* 글자 색상 및 배경색: 본문 강조용 컬러 피커
* 서식 지우기: 복사해서 붙여넣은 텍스트의 불필요한 인라인 스타일(HTML 태그)을 초기화하는 기능

## 2. 문서 구조 및 타이포그래피 (Structural)
EPUB는 목차(TOC) 생성이 중요하므로, 문서의 구조를 정의하는 도구가 반드시 필요합니다.

* 제목 스타일(Heading): H1, H2, H3, H4 선택 드롭다운 (자동 목차 생성과 연동)
* 본문 스타일: 일반 본문(Paragraph), 인용구(Blockquote) 블록 지정
* 리스트: 순서 있는 목록(<ol>), 순서 없는 목록(<ul>)
* 줄간격 및 자간: 가독성 조절을 위한 드롭다운 메뉴
* 들여쓰기/내어쓰기: 문단 시작 시 공백을 주기 위한 버튼 (CSS text-indent와 연동)

## 3. 멀티미디어 및 객체 삽입 (Insert Object)
전자책의 시각적 요소를 풍부하게 만드는 삽입 기능입니다. EPUB 내부에 파일이 올바르게 패키징되도록 처리해야 합니다.

* 이미지 삽입: 드래그 앤 드롭 업로드, 대체 텍스트(alt) 입력 필드, 이미지 크기 조절 및 정렬 기능
* 링크 삽입/제거: 내부 장(Chapter) 이동 링크 또는 외부 웹사이트 하이퍼링크
* 표(Table) 생성: 행/열 추가, 셀 병합 등 기본 표 에디터 (모바일 화면을 고려한 반응형 표 CSS 지원 필요)
* 구분선(HR): 장 내부에서 장면 전환 시 사용하는 수평선 삽입

## 4. EPUB 특화 도구 (EPUB Specific)
일반 웹 에디터와 차별화되는 전자책 전용 핵심 기능입니다.

* 페이지 나누기(Page Break): EPUB 리더에서 강제로 다음 페이지로 넘기도록 하는 CSS (page-break-before: always) 삽입 버튼
* 주석(Footnote/Endnote) 삽입: 팝업 주석이나 미주를 쉽게 만들 수 있는 링크-앵커 자동 연결 기능
* 특수문자 삽입: 문장 부호(—, …), 말줄임표, 외자 문장 부호 등 전자책 전용 기호 모음
* 코드 블록: IT/기술 서적 집필자를 위한 신택스 하이라이팅 기능

## 5. 집필 편의 기능 (Authoring Tools)
사용자의 작업 효율을 극대화하는 편의 도구입니다.

* 실시간 미리보기: 스마트폰, 태블릿, 전자책 단말기(E-ink) 비율별 화면 전환 뷰어
* 실시간 글자 수 세기: 공백 포함/제외 글자 수, 단어 수, 예상 독서 시간 표시
* 찾기 및 바꾸기: 원고 전체 또는 현재 장에서 특정 단어를 일괄 변경하는 기능
* HTML 소스 코드 보기: WYSIWYG 모드와 XHTML 소스 코드 모드 간 전환 (고급 사용자용)
